# Football Academy Player Management System

A **MERN Stack** web application to manage football academy players, track stats, and view achievements. Fully deployed on **Render** with a MongoDB Atlas backend.

---

## 🔥 Features

- **CRUD Operations**: Add, view, edit, and delete players.
- **Player Stats**: Calculate rating, market value, goals, assists.
- **Search & Filter**: Search players by name or filter by position.
- **Dashboard**: Home page shows total players, average rating, and total market value.
- **Achievements**: Highlights top scorer, best-rated player, most valuable, and most assists.
- **Responsive UI**: Works on desktop and mobile.

---

## 🛠 Technology Stack

- **Frontend**: React.js, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Deployment**: Render (Frontend & Backend)
- **Notifications**: react-hot-toast
